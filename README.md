# Real-Estate-Price-Prediction
An end-to-end Machine Learning project that predicts house prices in Bengaluru using property features such as location, total square feet, number of bathrooms, and BHK. The project covers data cleaning, feature engineering, model training, hyperparameter tuning, evaluation, and deployment.
